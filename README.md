# ⚛️ React - Hora de Codar

Projeto Cost é uma página SPA que tem como objetivo criar e gerenciar projetos e seus custos.

A página realiza CRUD que consome uma fake API com o Json-Server

### Para executar o projeto: 

* para baixar as dependências:
    ```
    npm install
    ```

* para executar o front-end:
    ```
    npm start
    ```

* para executar o back-end:
    ```
    npm run backend
    ```
